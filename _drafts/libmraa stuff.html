--- layout: blog title: lib mraa stub --- __libmraa__ Intel's Low Level Skeleton Library for IO Communication on GNU/Linux platforms can be found [here](https://github.com/intel-iot-devkit/mraa). An example Python application that interacts with the [SparkFun 9 Degrees of Freedom Block for Edison](https://www.sparkfun.com/products/13033) can be found [here](https://github.com/smoyerman/9dofBlock). You can monitor our progress on a Java equivalent to that example [here](https://github.com/jeffplourde/9dofBlockJava).
