Live Data Streaming from the MD PnP Lab

Demo Web App

Press Start to subscribe to data from the specified domain, topic, and partition. Currently only numeric and waveform data will be represented on the screen. Press Stop when you're done to save your computer and the server a lot of wasted time. You will see the interval from 12 seconds ago to 2 seconds ago.
