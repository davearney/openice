API Access to the MD PnP OpenICE Network

This website utilizes a JavaScript API to access the MD PnP medical device test bed. The purpose of this site is to visualize the vital sign data flowing from the OpenICE network in the MD PnP lab in Cambridge, MA.

To start the sample application, please navigate to the demo page.


White House Smart America Healthcare App Challenge

Have an idea for an innovative healthcare application but lack the data and resources necessary to develop it?

MD PnP is holding an app challenge to enable healthcare innovation outside the walls of hospitals and medical device manufacturers. Contestants will use websockets to access real-time streaming medical device data from the MD PnP test lab to their applications.

OpenICE.js Documentation

The files used to visualize data on this webpage are listed below. The detailed documentation of OpenICE.js can be found by clicking the link below.

Files in the Repository

openice.js
The JavaScript API developed at MD PnP used to access data flowing from the lab
openice.flot.js
The example script used in the demo webpage to pass data from OpenICE.js to a visualization library called flot.js
jquery.flot.js
The flot script used for plotting the vital signs data originating from the MD PnP lab
jquery.flot.time.js
The script used for creating 'flots' with a time-based axis
jquery.js
A dependency of several scripts
bootstrap.css
A front-end framework used to improve site layout and appearance
