Smart America Demo JS API

Javascript API Access to the MD PnP OpenICE Network

This website utilizes an API to access the MD PnP medical device test bed. The purpose of this site is to visualize the vital sign data flowing from the OpenICE network in the lab in Cambridge, MA.

To start the sample application, please navigate to the demo page.


View the Demo


Download the Javascript API and Sample App from GitHub

The git repository for the MD PnP Lab javascript API, this website, the 'flot' data visularization script and supporting assets are available on GitHub. Please contribute to this project by forking this code and submitting a pull request.

OpenICE.js Documentation

The files used to visualize data on this webpage are listed below. The detailed documentation of OpenICE.js can be found by clicking the link below.

Files in the Repository

openice.js
The javascript API developed at MD PnP used to access data flowing from the lab
openice.flot.js
The example script used in the demo webpage to pass data from OpenICE.js to a visualization library called flot.js
jquery.flot.js
The flot script used for plotting the vital signs data originating from the MD PnP lab
jquery.flot.time.js
The script used for creating 'flots' with a time-based axis
jquery.js
A dependency of several scripts
bootstrap.css
A front-end framework used to improve site layout and appearance
